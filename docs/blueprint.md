# **App Name**: DevFolio

## Core Features:

- About Me Section: A section providing an introduction and background information about the developer.
- Projects Section: Showcase of software development projects with descriptions and GitHub links.
- Skills Section: Categorized display of programming languages, tools, and soft skills.
- Contact Section: A section with contact information and a contact form for potential employers or collaborators.
- Interactive Project Cards: Visually engaging project cards with clickable links to the GitHub repositories and live demos (if available).

## Style Guidelines:

- Dark background with a subtle deep blue accent: Background color: #121C29
- Primary color: Light blue (#ADD8E6) for highlighting key elements and interactive components.
- Accent color: A brighter shade of blue (#87CEEB) to emphasize calls to action and important links.
- Font pairing: 'Inter' (sans-serif) for body text, 'Space Grotesk' (sans-serif) for headings.
- Simple, minimalist icons for navigation and skills representation. The icons must be monochromatic to integrate in a dark-themed interface. 
- Clean and structured layout with clear sections and consistent spacing.
- Subtle transition animations when navigating between sections or when hovering over project cards to indicate interactivity.
